# Eco-Commerce Recommender: The Green-Product Engine 🌿

A data science project focused on merging **Recommender Systems** with **ESG (Environmental, Social, and Governance)** metrics to help consumers make sustainable choices.

## 🚀 The Goal
To build a system that identifies "Green" products and recommends them as alternatives to conventional items, using NLP to verify sustainability claims and filter out greenwashing.

## 🛠️ Tech Stack
- **Languages:** Python
- **Libraries:** Pandas, Scikit-Learn, BeautifulSoup (Scraping), Transformers (NLP)
- **Deployment:** Streamlit (planned)
